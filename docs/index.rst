Welcome to the MOSAICS documentation
====================================

About
-----

MOSAICS (*include exact acronym*) is a Python package which uses high-resolution two-dimensional template matching for identifying agreement and disagreement between experimental data and a reference template.


Installation
------------

Mosaics is currently source-install only; follow the instructions on the `project's GitHub page <https://github.com/Lucaslab-Berkeley/MOSAICS>`_.
